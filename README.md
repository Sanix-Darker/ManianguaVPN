# Description:

Simple script python To make a VPN's connexion to vpnbook.

## Installation Linux:
```shell
$ git clone https://github.com/Sanix-Darker/ManianguaVPN

$ cd ManianguaVPN 

$ python main.py
```

Then go at https://www.vpnbook.com/freevpn, and get username and password for credentials.
